This folder contains extra resources for the odometry, 3D field, and joystick views. For more details, see the "Custom Fields/Robots/Joysticks" page in the AdvantageScope documentation (available through the documentation tab in the app or the URL below).

https://github.com/Mechanical-Advantage/AdvantageScope/blob/main/docs/CUSTOM-CONFIG.md

To be able to use the 3D model, move the contents of this folder to the folder presented when you press Help>Show FRC Data Folden in AdvantageScope.
